### Taxonomy of _Phythoptora ramorum_
> Clade: 	SAR \
Phylum: 	Oomycota \
Order: 	  Peronosporales \
Family: 	Peronosporaceae \
Genus: 	  Phytophthora \
Species: 	_P. ramorum_

There are 4 clonal lineages:
> NA1 - North America 1 \
NA2 - North America 2 \ 
EU1 - Europe 1 \
EU2 - Europe 2
Note: Name based on origin and time of isolation
